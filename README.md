## 🛠️ Technical Ecosystem

### Core Layout & UI
*   **Semantic HTML5 & Modern CSS3:** Architected completely with vanilla CSS custom properties (variables) for strict, atomic design maintenance.
*   **Glassmorphism UI Patterns:** Implemented a modern backdrop-filtering navbar layout that remains perfectly fluid across mobile and desktop breakpoints.
*   **Dynamic Theme Engine:** Pure JavaScript toggle handler allowing instant switches between standard and highly contrastive Dark modes.

### Performance & Engine
*   **Zero-Dependency Scripting:** Uses native browser APIs exclusively—no external UI frameworks or heavy overhead libraries.
*   **Intersection Observer API:** Asynchronous lazy-triggering scroll visibility thresholds (`.fade-in`) to keep frame rendering lightweight.
*   **Vanilla Node.js Backend Server:** Custom HTTP routing utility created from scratch to handle asset pipeline mime-types efficiently without Express boilerplate.
